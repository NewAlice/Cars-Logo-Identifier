# Cars-Logo-Identifier
Cars Logo Identifier is an Deep Learning Model based on Convolutional Neural Networks. This CNN model is trained over 20,778 50x50px RGB images depicting logotypes of 40 different car brands. This Dataset of 20,778 images was handpicked from loosely scraped websites by Kuba Siekierzy≈Ñski and contains images, photos, drawings, sketches at various color schemes (black/white, RGB, CMYK, one-color), different angles and can sometimes contain some noise (other logotypes, background, etc.).

# Dataset:-
Images (20778 files, 54 MB - ZIP) for the train and test set - http://u.42.pl/GEt7_images
